# 🌪️ Hexado Chasing V2

A high-performance, browser-based 3D tornado-chasing simulation built by **Hexagon Studio**.

This project is a technical demonstration of procedural 3D generation using **Three.js (r128)** and Vanilla JavaScript. We are currently in the **Alpha Development** phase.

---

## 🏢 About Hexagon Studio
We are an indie development studio focused on creating unique, code-driven experiences. This project is our current lead technical challenge, focusing on math-heavy weather systems and optimized game loops.

---

## 🚀 Technical Highlights

* **14-Module Architecture:** Completely decoupled systems using a global `EventBus`.
* **Zero-Asset Pipeline:** 100% of the game world (trees, barns, tornado, clouds) is generated via JavaScript math—no external 3D files.
* **Unified Terrain:** All physics and visuals are synced to a "Single Source of Truth" height function.
